# Youtube API

2020.11.10



- [Google API Console](https://console.developers.google.com/)에 접속한다.
- Google 아이디 로그인
- `YouTube Data API v3` 검색 후 다운로드
- 왼쪽 상단 `Google APIs` 로고 옆에 프로젝트를 생성한다.
- 메뉴 > API 및 서비스 > 사용자 인증 정보 > API 발급
- 발급된 API 키 `이름` 클릭
- 세부사항에서 `선택한 API`에서 `Youtube Data API v3` 선택 후 저장
