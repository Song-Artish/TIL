## 시스템

> 웹 기반 **EDC(Electronic Data Capturwe) 시스템**

- EDC, 즉 데이터를 입력하는 시스템
- 시스템 내부에 data validation을 위한 **ECS**(Edit Check System)을 포함한다.

**입력 데이터**

|  visit  |                          설명                           |
| :-----: | :-----------------------------------------------------: |
| **V1**  | 각종 정보를 받고 해당 시험자가 적합한지를 판별하는 과정 |
| **V2**  |            다시 방문하면, 무작위 배정을 진행            |
| **V3~** |         이후 매 방문마다 투약약물의 효과를 확인         |
