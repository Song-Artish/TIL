## Web Socket

> 하나의 TCP 접속에 전이중 통신 채널을 제공하는 컴퓨터 통신 프로토콜

- 실시간 (양방향) 통신을 하기 위해 사용한다.



## NPM에서의 사용

- 먼저 `web socket` 폴더에서 필요한 node package를 설치한다.

```bash
npm i
```

- 다음으로 `nodemon` 클라이언트를 글로벌로 설치한다.

```bash
npm i nodemon -g
```

- 그리고 nodemon을 클라이언트로 웹 소켓 파일을 실행한다.

```bash
nodemon <파일명>.js
```
